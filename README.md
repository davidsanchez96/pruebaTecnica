Quick Start
-----------
```shell
$ git clone git@github.com:davidsanchez96/pruebaTecnica.git
$ cd pruebaTecnica
$ npm install
$ npm run dev
$ visit localhost:3000
```

Commands
--------

|Script|Description|
|---|---|
|`npm run dev`| Run development server with webpack-dev-server @ `localhost:3000`|
